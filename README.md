# projeto 002 - gerenciamento
projeto de gerenciamento de biblioteca, feito com HTML, CSS, Python e SQLite

Obs: 
Bibliotecas/Extensões ultilizadas:
  - tkinter (interface em python)
  - flask (framework, para o html)
  - sqlite3 (banco de dados para ambos)

No "...(tkinter)" rodar o "view.py" depois o "main.py".
No "...(html)" rodar o "view.py" depois o "app.py" e acessar o link que aparece no terminal no navegador.
